# Daejeon Travel Lounge(DTL) 홈페이지
<h2>Skills</h2>
· 개발도구 : Eclipse <br/>
· 사용언어 :  Java / Spring / JSP / HTML / CSS / JS / SQL Developer / MyBatis / Bootstrap
<h2>Features</h2>
· 회원가입 / 로그인 기능 구현 <br/>
· Spring Security 클래스를 이용한 비밀번호 암호화 기능 구현 <br/>
· 사용자(USER)와 관리자(ADMIN) 구분하여 기능 구현 <br/>
· 기본적인 CRUD 구현
